About IWTBA

I Want To Be A (IWTBA) is learning aid developed by Mark Ally. Describe what you want, and let it show you the way. You can find code and a more detailed description on GitHub.

Courses come from Coursera, Job postings from NYC Gov't and Github Jobs

The page theme is Grayscale, courtesy of Start Bootstrap. It features stock photos by Gratisography.
